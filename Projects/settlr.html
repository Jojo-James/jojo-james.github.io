Settlr is a modern Android expense-sharing application inspired by Splitwise, designed to simplify group expense management and bill splitting. The application enables users to create groups, record shared expenses, manage balances, and track financial transactions through an intuitive and responsive user interface built with Jetpack Compose and Material Design 3.

Developed using a clean MVVM architecture, Settlr combines the reliability of Room Database for offline-first functionality with Firebase services for authentication and future cloud synchronization. The application ensures users can continue managing expenses even without an internet connection while maintaining a scalable architecture for future online features. The project emphasizes maintainability and scalability through a repository pattern that abstracts data sources from business logic. This design enables seamless integration between local storage and cloud services while providing a smooth user experience with animated screen transitions and responsive UI components.

Settlr is designed as a production-ready foundation for a complete expense-sharing platform, supporting future enhancements such as real-time synchronization, notifications, settlement tracking, and collaborative group management.
